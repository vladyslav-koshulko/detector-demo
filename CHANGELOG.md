# Історія оновлень: Vision Intelligence System

## [v5.1] — Поточна версія
### Додано
- **Batch Folder Detection (Desktop):** Обробка всіх файлів у папці (локальна/мережева) з сортуванням за часом модифікації.
- **ZIP Export:** Збереження оверлеїв (bbox) в архіві `model_name_YYYYMMDD_detected.zip`.
- **Expanded Sources:** Підтримка папок, розширених форматів відео/зображень у web та desktop.
- **Model Manager (Web):** Завантаження моделей через URL або upload файлу.

### Виправлено
- **Web Auth:** Відновлено дефолтні користувачі при порожньому `web_users_auth`.
- **Desktop Grid UX:** Вікна камер створюються лише при активному стрімі.

---

## [v5.0 Ultimate]
### Додано
- **Software Zoom (PTZ):** Можливість цифрового наближення через контекстне меню (права кнопка миші на вікні камери).
- **Auto-Scroll Trigger:** Автоматичне фокусування та прокрутка матриці до вікна, де виявлено рух або об'єкт (person/car).
- **Dynamic Model Downloader:** Завантаження нових моделей YOLO прямо з інтерфейсу (через випадаючий список моделей).
- **Advanced Grid Engine:** Можливість вибору кількості потоків на рядок (від 1 до 4) з миттєвим перешиковуванням сітки.
- **Role-Based Access (RBAC):** Розділення прав на "Admin" (керування джерелами) та "View" (тільки перегляд) для веб-користувачів.
- **Mobile-First Web UI:** Повністю адаптивний дизайн для смартфонів з великими елементами керування.

### Виправлено
- **KeyError [idx]:** Повністю усунено помилки ініціалізації двигунів при динамічному додаванні вікон.
- **Boxes Attribute Error:** Виправлено сумісність з бібліотекою `supervision` для кастомних ваг моделей.
- **Grid Ordering:** Вікна камер тепер заповнюються послідовно, починаючи з першого верхнього лівого кута.
- **Qt Plugin Conflict:** Виправлено запуск на Linux системах через примусове налаштування шляхів до плагінів Qt.

## [v4.0] — Адміністрування та Аналітика
### Додано
- **Night Boost (CLAHE):** Адаптивне покращення контрасту для нічних зйомок.
- **Object Visibility Filter:** Чекбокси для кожного класу об'єктів (можна вимкнути детекцію "собак" або "сумок", залишивши тільки "людей").
- **User Management Table:** Інтерактивна таблиця в адмінці для керування доступами (Allow/Block/Delete).
- **Infinite Scroll Matrix:** Можливість додавати необмежену кількість камер завдяки вертикальній прокрутці.

## [v3.0] — Qt Migration
### Додано
- Перехід з Python-скрипта на повноцінний десктопний додаток на базі **PySide6 (Qt)**.
