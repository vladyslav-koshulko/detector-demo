# Smart Analysis Server: Демонстраційний Стенд для Відеоаналізу

Цей проєкт є демонстраційним стендом для аналізу відеопотоку в реальному часі з використанням різних моделей машинного зору та алгоритмів обробки зображень.

## Суть Проєкту

Проєкт призначений для порівняння роботи популярних моделей детектування об'єктів та візуалізації різних аналітичних шарів відеопотоку. Він дозволяє "наживо" побачити різницю в точності та швидкості роботи різних архітектур нейромереж.

## Що Всередині:

1.  **Детектування Об'єктів:**
    Використовуються три сучасні архітектури:
    *   **YOLOv8** та **YOLOv10** (від Ultralytics) — швидкі та популярні моделі.
    *   **RT-DETR** — Real-Time Detection Transformer.
    *   **RF-DETR** — спеціалізована реалізація детектора.

2.  **Відеоаналітика (OpenCV):**
    Для кожного кадру паралельно обчислюються:
    *   **Optical Flow:** візуалізація векторів руху.
    *   **FG Mask:** виділення рухомих об'єктів на статичному фоні.
    *   **Sobel Filter:** виділення контурів об'єктів (edges).
    *   **Heatmap:** тепловая карта інтенсивності зображення.

3.  **Інтерфейс ("Smart Analysis Server"):**
    *   Скрипт `detection.py` збирає всі результати в єдину багатопанельну сітку.
    *   Підтримується **зум** (`=` / `+` для збільшення, `-` для зменшення) та **вертикальне прокручування** (`W` для вгору, `S` для вниз), що зручно для детального вивчення кадров при великій кількості аналітичних шарів. Натисніть `R` для скидання зуму та прокрутки.

4.  **Артефакти:**
    *   У корені знаходяться ваги моделей (`.pt`, `.pth`).
    *   Логи (`.log`) містять статистику FPS та результати роботи алгоритмів для подальшого аналізу.

## Інструкції щодо Запуску

### Передумови

*   **Python 3.x**
*   **Вебкамера** (для захоплення відеопотоку)
*   **Файли моделей:** Переконайтеся, що файли `yolov8n.pt`, `rtdetr-l.pt`, `yolov10n.pt` та `rf-detr-base.pth` знаходяться в кореневій директорії проєкту.

### Налаштування Віртуального Оточення та Встановлення Залежностей

Для ізоляції залежностей рекомендується використовувати віртуальне оточення:

1.  **Створіть віртуальне оточення:**
    ```bash
    python -m venv .venv
    ```

2.  **Активуйте його:**
    *   На Windows:
        ```bash
        .venv\Scripts\activate
        ```
    *   На Linux/macOS:
        ```bash
        source .venv/bin/activate
        ```

3.  **Встановіть необхідні бібліотеки:**
    ```bash
    pip install -r requirements.txt
    ```
    *Примітка: Для встановлення `torch` з підтримкою GPU (CUDA), будь ласка, зверніться до [офіційної документації PyTorch](https://pytorch.org/get-started/locally/).*

### Запуск Проєкту

Переконайтеся, що ваше віртуальне оточення активоване, і запустіть скрипт:

1.  **Запустіть головний скрипт:**
    ```bash
    python detection.py
    ```

2.  **Керування:**
    *   Натисніть `Q` для виходу з програми.
    *   Використовуйте `W` та `S` для вертикальної прокрутки.
    *   Використовуйте `=` або `+` для збільшення масштабу (zoom in).
    *   Використовуйте `-` для зменшення масштабу (zoom out).
    *   Натисніть `R` для скидання масштабу та прокрутки до початкових значень.
